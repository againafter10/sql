Generating a hierarchical list:
1. we first need to retrieve root employee, i.e. top level employee,
2. after the root level is retrieved, we need to find all child nodes for that root level,
3. perform step 2 until all levels are retrieved.

This can be achieved using a Recursive Common Table Expression, which consists of two parts:
1. Retrieve root level – this is known as Anchor in a Recursive CTE
2. retrieve all children of root level – this is known as Recursive Part in a Recursive CTE.


eg:
WITH EmployeeHierarchy(empno, ename, job, EmpLevel) AS ( SELECT empno,ename,job, 0 AS EmpLevel FROM emp WHERE mgr IS NULL UNION ALL SELECT emp.empno, emp.ename, emp.job, EmpLevel + 1 FROM emp INNER JOIN EmployeeHierarchy AS EmpH ON emp.empno = EmpH.empno) SELECT empno,ename,job,EmpLevel FROM EmployeeHierarchy ORDER BY empno and EmpLevel < 3;
